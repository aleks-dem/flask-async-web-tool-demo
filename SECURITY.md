# Security Policy

## Supported Versions

This repository is a public demo project. Security fixes are applied to the latest `main` branch only.

## Reporting a Vulnerability

Do not open public issues for security-sensitive findings.

Please report privately with:
- A clear description of the issue.
- Reproduction steps.
- Impact assessment.
- Suggested fix (optional).

If you cannot contact privately, open an issue with minimal details and mark it as security-related so maintainers can move discussion off-thread.

## Scope Notes

- This repository is intentionally sanitized and runs in mock mode by default.
- No production credentials or production infrastructure are included.
- Security hardening focuses on code quality suitable for portfolio/public demo usage.
